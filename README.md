Yii2-outlog
===================
Yii2 component for submiting errors to outer logging system

Installation
---
Run
```code
php composer.phar require "demi/yii2-outlog" "~1.0"
```
or


Add to composer.json in your project
```json
{
	"require": {
  		"demi/yii2-outlog": "~1.0"
	}
}
```
then run command
```code
php composer.phar update
```
